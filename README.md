## 📖 项目简介

**神龙剧本编辑器** 是一款专为 AI 视频生成打造的辅助工具，帮助创作者理清剧情思路、设计情节发展。无论你是短视频创作者、编剧还是 AI 视频爱好者，都能通过本工具将创意转化为结构清晰的剧本。

### 🎯 核心价值

- 🎬 **AI 视频生成辅助**：为 AI 视频创作提供清晰的剧本结构
- 📝 **剧情思路梳理**：可视化方式组织故事脉络
- 💾 **专用格式支持**：使用 `.sljb`（神龙剧本）格式保存和分享

## ✨ 主要特性

### 📋 剧本管理
- **新建剧本**：快速创建空白剧本，从零开始构思
- **导入剧本**：打开已有的 `.sljb` 格式文件继续编辑
- **保存剧本**：将创作内容保存为专用剧本格式

## 🚀 快速开始

### 本地使用
1. 下载本仓库
   ```bash
   git clone https://github.com/Spacedeletion/SL-Playbook-Editor.git
   ```
2. 浏览器打开 `神龙剧本编辑器.html` 文件
3. 开始创作你的剧本！

## 📁 文件格式

### .sljb（神龙剧本）格式说明
本软件使用自定义的 `.sljb` 格式保存剧本，包含：
- 剧本元数据（名称、作者、创建时间）
- 核心主题
- 情感弧线
- 风格统一指令
- 段落与分镜编排
- 后期制作与系列规划
